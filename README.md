## Introduction

Socket programming is a way of connecting two nodes on a network to communicate with each other. This repository aims to provide examples and projects that help understand the workings of socket programming.

## Projects

1. **Chat Application**  
   A simple chat application that allows users to communicate in real-time through sockets.

2. **File Transfer Protocol**  
   An application that demonstrates how to transfer files between systems using sockets.

3. **HTTP Server**  
   A basic implementation of an HTTP server that responds to GET requests.

4. **Multi-threaded Server**  
   An example of a server that handles multiple clients using threads.

## Getting Started

To get started with the projects:

1. **Clone the repository**:  
   ```
   git clone https://github.com/MH165/sockets.git
   ```

2. **Navigate to the project folder**:  
   ```
   cd sockets
   ```

3. **Follow the documentation in each project folder** to set up and run the applications.
